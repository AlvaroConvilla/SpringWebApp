# SpringWebApp
A simple web application Hello World, using Maven + Spring. Serving Web Content with Spring MVC

### Instalar Maven
```
sudo apt install maven
```

### Compilar con Maven
```
mvn compile
```

### Crear fichero JAR 
```
mvn package
```

### Ejecutar aplicaci√≥n web
```
java -jar target/helloWorld_v1.1-0.1.1.jar 
```
